# Blackjack Game

A simple text-based Blackjack game written in Python. You play in the terminal against a dealer using a standard 52-card deck.

## Features

- Standard 52-card deck
- Shuffled cards each game
- One player vs dealer
- Player can hit or stand
- Dealer draws until reaching at least 17
- Aces are handled as 1 or 11 automatically
- Win, loss, and push outcomes

## How to Run

1. Make sure you have Python 3 installed.
2. Open a terminal in this project folder.
3. Run:

```bash
python3 blackjack.py
```

## How to Play

- At the start, you will see both of your cards and one of the dealer's cards.
- Enter `h` to hit and take another card.
- Enter `s` to stand and end your turn.
- The dealer then draws until the hand value is at least 17.
- The game announces whether you win, lose, or push.
